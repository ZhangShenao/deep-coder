# Deep-Coder 项目上下文

## 1. 项目概述

### 1.1 项目背景
Deep-Coder 是一款基于 LangGraph 1.0+ 框架开发的智能编程助手。它能够根据用户需求自动生成代码，并在安全的沙箱环境中运行和调试，帮助开发者快速实现想法。

### 1.2 核心功能
- 自然语言描述需求，自动生成代码
- 代码在隔离沙箱中安全执行
- 实时展示执行过程和结果
- 支持代码迭代优化和调试
- 会话状态持久化保存

### 1.3 技术栈
| 类别 | 技术选型 |
|------|----------|
| 后端语言 | Python 3.12 + uv |
| Agent 框架 | LangGraph 1.0+ |
| LLM 模型 | 智谱AI GLM-4-Flash |
| 沙箱方案 | E2B Code Interpreter |
| Web 框架 | FastAPI + Uvicorn |
| 前端框架 | Streamlit |
| Checkpointer | InMemorySaver（初期）|

### 2.2 架构分层说明

| 层级 | 组件 | 职责 |
|------|------|------|
| **Frontend Layer** | Streamlit UI | 用户交互、代码展示、执行结果可视化 |

| **Sandbox Layer** | E2B | 代码安全执行、环境隔离、文件管理 |

## 3. 核心模块划分

```
deep-coder/
├── backend/
│   ├── agent/          # LangGraph Agent、提示词、工具
│   ├── api/            # FastAPI 路由、WebSocket 处理
│   ├── sandbox/        # E2B 沙箱生命周期管理
│   ├── checkpoint/     # 会话状态持久化
│   ├── models/         # Pydantic 请求/响应模型
│   ├── config/         # 配置和设置
│   └── main.py         # FastAPI 应用入口
├── frontend/
│   ├── app.py          # Streamlit 主应用
│   └── utils/          # API 客户端、会话工具
├── Dockerfile          # 后端 Dockerfile
├── Dockerfile.frontend # 前端 Dockerfile
├── docker-compose.yml  # Docker Compose 配置
├── pyproject.toml      # uv 项目配置
└── deploy.sh           # 一键部署脚本
```

### 3.1 Agent Core 模块
- **文件**: `backend/agent/core.py`
- **职责**: LangGraph Agent 核心配置，使用 `create_react_agent` 创建 ReAct 风格的 Agent

### 3.2 Sandbox Manager 模块
- **文件**: `backend/sandbox/manager.py`
- **职责**: E2B 沙箱生命周期管理（创建、获取、销毁）

### 3.3 API 模块
- **文件**: `backend/api/routes/`
- **职责**: 聊天 API、沙箱管理 API、会话管理 API

## 4. 重要提示

**代码生成优先使用 Context7 MCP 工具**

当生成代码时，如果明确需要使用具体的 SDK 或 API，**必须优先使用 `context7` MCP 工具**获取最新的 SDK 用法及官方文档，确保代码示例的准确性和时效性。

示例用法：
```
# 先解析库 ID
mcp__context7__resolve-library-id(libraryName="langgraph", query="create_react_agent")

# 再查询文档
mcp__context7__query-docs(libraryId="/langchain-ai/langgraph", query="how to create react agent with system prompt")
```
